# stuMap

## 项目介绍
该项目是基于图的最短路径算法实现的一个GUI图形界面。
该项目为学校的软件算法综合设计二级项目------校园行走最优路径查询系统。

## 项目使用
本项目为一个java包，直接拖入编译器的已有项目或者空项目即可，运行Main.java即可启动程序。

## 项目功能
输入起点、终点、添加途经地点，点击获取最短路径，即可获得一条最短路径。
点击显示所有路径即可显示所有路径。
